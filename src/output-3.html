Requisitos e Etapas

Do Cliente:

  • Disponibilizar dados, informações e documentos necessários à realização dos serviços aqui estipulados. Ficando ciente que atrasos relativos a entrega do que for solicitado pode acarretar atraso dos prazos acima estipulados;
  • Fornecer espaço físico adequado quando houver necessidade de visitas na fábrica;
  • Disponibilizar equipe com pelo menos 1 colaboradores para acompanhamento do projeto e validação de resultados;
  • Disponibilizar todos os dispositivos e materiais necessários no desenvolvimento do sistema.
  • Intermediar contato com equipe de campo.
  • Disponibilizar base de dados e informações relacionadas aos treinamentos em ergonomia realizados em ambiente físico.
  • Disponibilizar sala para instalação do ambiente de treinamento.
  • Fornecer um servidor para hospedagem do sistema

Do Proponente:

  • Garantir a confidencialidade das informações fornecidas pela empresa;
  • Apresentar as soluções propostas ao cliente;  
  • Disponibilizar Consultor (es) capacitado (s) para o trabalho;  
  • Definir, em comum acordo com o responsável da empresa, as datas e horários para a realização das atividades propostas;
  • Disponibilizar para cliente toda documentação do projeto;
  • Desenvolvimento do ambiente virtual;
  • Trabalhar conteúdo disponibilizado pelo contratante;
  • O conteúdo será trabalhado em formato interativo;
  • Dar suporte técnico a partir da primeira entrega e por 3 meses após entrega no quinto mês;
  • Disponibilizar build do sistema para executar nos servidores web Apache ou IIS;
